### 프로젝트 소개

#### 주제
주식 초보를 위한 증권 정보 제공 Web Service

#### 기술 스택
-  Springboot
-  JAVA 17
-  Gradle
-  Docker

#### 클라우드 서비스
- AWS RDS - MySQL 8 버전
- AWS ECR(Docker Image Registry)
- AWS EC2 - Ubuntu
- AWS Route53
- AWS Certificate Manager
- AWS ElastiCache
